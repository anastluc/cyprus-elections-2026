# Predict

Have a big "Predict button" on the top that takes you to the Predict page

There you are prompted to give the percentage results of each party

A histogram is created that has a timestamp and a name and can be shared to social media (facebook, X, Linkedin, ) and a permanent link to this prediction. Target is for people to be able to claim (after the elections) here "I told you so"

Leaderboard (after the election results):

You explain how the scoring will be: 
the absolute difference of each party prediction percentage to the actual is added
Provide an example 

No prizes or anything, just endless Kudos!

Can also predict other things:
- total number of parties securing entry to parliament
- participation percentage 
- over/under a given line

Advanced mode:
- to predict the actual names of members elected on each district
- percentage of each party in each district
